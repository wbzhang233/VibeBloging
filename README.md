# VibeBloging

借助 AI 实现氛围感技术博客写作与归档。

## 项目简介

VibeBloging 是一个以 AI 为核心辅助工具的技术博客项目，目标是高效、有氛围感地创作和归档技术内容。

## 目录结构

```
VibeBloging/
├── _project/          # 项目管理（memory、log、规范）
├── AiweQuant/         # AI/Agent 在量化投研领域的应用
└── README.md
```

## 博客方向

- **AiweQuant** — AI/Agent 系统在量化投研中的应用与实践

## 工作流

1. 确定主题与大纲
2. AI 辅助草稿生成
3. 人工润色与校对
4. 归档到对应目录
