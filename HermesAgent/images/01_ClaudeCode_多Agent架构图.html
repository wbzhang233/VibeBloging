Claude Code
多 Agent 编排架构

Anthropic 官方打造的专业级端侧执行智能体。Coordinator 主代理统筹调度，多个子代理并行执行，每个子代理运行在独立 Git Worktree 隔离沙箱中，实现工程级精准落地。

记忆层次架构
L1
项目记忆（CLAUDE.md）
当前项目全局规则与上下文，每次会话自动加载，人工手动维护
PROJECT
L2
用户记忆（~/.claude/claude.md）
个人操作偏好与长期规则，跨项目持久化，人工配置
USER
L3
会话记忆
单次会话上下文，/resume 命令跨会话恢复，会话结束后 auto-dream 异步整合
SESSION
L4
团队共享记忆（Agent Teams）
多代理协作时共享任务上下文与协作规则，通过 SendMessage 协议同步
TEAM
🔒
Git Worktree 隔离
每个 Sub-Agent 在独立 git worktree 中运行，彻底避免上下文污染，支持工具级沙箱权限管控
📡
SendMessage 协议
子代理之间通过 SendMessage 协议直接通信，共享带依赖追踪的任务列表，支持异议协调
🖥️
Computer Use
macOS 全桌面控制能力，支持点击、拖拽、截图、跨应用操作，实现真正端侧自动化
🔌
MCP 协议
Model Context Protocol 原生支持，无限扩展第三方工具与服务，内置 45+ 官方工具
📱
Dispatch 移动端
通过 Dispatch 实现移动端远程控制，跨设备管理运行中的 Agent 任务
🔐
精准权限管控
细粒度权限审批机制，危险操作需用户确认，企业级合规与成本跟踪
