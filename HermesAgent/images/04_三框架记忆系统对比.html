选型决策指南
🔵 选 Claude Code
工程级代码开发场景
你是专业开发者，需要 AI 帮助完成复杂的软件工程任务，对代码质量和工程规范有严格要求。
  • 大型代码库重构、多文件联动开发
  • 需要并行执行多个子任务
  • 要求企业级权限审批和安全合规
  • 重度依赖 MCP 生态和 Computer Use
🟢 选 OpenClaw
全天候个人 AI 助理场景
你需要一个能通过 WhatsApp/Telegram 随时联系、24小时在线、会主动"心跳"自检的个人 AI。
  • 需要 IM 平台（WhatsApp/Discord）集成
  • 希望 Agent 每小时自动检查任务状态
  • 预算极低，$5 VPS 可满足需求
  • 需要跨会话持久记忆的个人助理
🟣 选 Hermes Agent
自进化智能体研究场景
你关注 Agent 的自我进化能力和个性化适配，希望 Agent 越用越懂你，且完全本地私有化部署。
  • 重视 Agent 个性化和用户建模能力
  • 希望 Agent 自动从经验中学习（Skill 蒸馏）
  • 需要本地 Ollama 推理，数据完全私有
  • AI Agent 研究者，关注自进化架构
