单次工具调用 · 完整执行管道

1
XML/JSON 解析
从模型输出中提取 tool_name, tool_id, input_params
PARSE
2
权限检查
Global → Project → Runtime 四级规则链
ALLOW / DENY / NEEDS_APPROVAL
PERMISSION
3
PreToolUse Hook
执行所有匹配的 PreToolUse 脚本
exit 2 → 阻止工具执行
HOOK
4
工具执行
路由到对应执行器（文件系统 / Shell / 网络 / Agent）
捕获 stdout, stderr, exit_code
EXECUTE
5
PostToolUse Hook
执行所有匹配的 PostToolUse 脚本
可读取 / 修改执行结果
HOOK
6
生成 tool_result block
包含 tool_use_id + content（文本 / 图片 / 错误）
INJECT
7

触发下一轮 claude_api() 推理
LOOP
