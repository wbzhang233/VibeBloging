Worktree 生命周期

+
创建 worktree
EnterWorktree(name="feature-x")
git worktree add .claude/worktrees/feature-x -b branch/x
↻
子代理独立执行
所有文件修改在独立分支，不影响 main
⇌
主代理审查并合并
git diff worktree/x → cherry-pick / merge
×
清理 worktree
ExitWorktree(action="remove")
git worktree remove + git branch -d
