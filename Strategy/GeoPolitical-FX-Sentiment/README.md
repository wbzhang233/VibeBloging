# GeoPolitical-FX-Sentiment
## 地缘政治风险下的外汇与贵金属高频策略研究

> **背景**：2026 年 4 月，美伊军事对峙、BoJ 正常化、特朗普关税滞胀三重矛盾交织
> **适用对象**：商业银行资金营运中心，外汇与贵金属交易台
> **资产范围**：G7 外汇 + XAU/USD、XAG/USD
> **数据源**：路透社新闻（Reuters Newswire）+ LLM 情感/情景分析

## 三套策略对比

| 编号 | 策略 | 核心方法 | 信号频率 | LLM 角色 |
|------|------|---------|---------|---------|
| 02 | NSIF 舆情驱动 | 逐条新闻 → NSIF 因子 → 3 层信号过滤 | 高频（分钟级） | 情感提取器 |
| 03 | 宏观情景多因子 | NSI + 跨资产领先滞后 + 状态机 + BoJ Tracker | 中频（15 分钟） | 宏观策略师 |
| 04 | 第一性原理策略集 | 5 个独立策略并行（行为金融/资产定价/物理/复杂系统/NLP） | 混合 | 视策略而定 |

### 04 策略集内五个子策略

| 子策略 | 理论基础 | 核心机制 |
|--------|---------|---------|
| A 过度反应-回归 | 行为金融学（归纳） | 3σ 冲击 → LLM 分类 one_shot/structural → 逆向入场 |
| B 避险相对价值 | 资产定价（演绎） | 危机类型 → 避险排序 → 配对交易最强/最弱 |
| C 新闻流物理学 | 信息动力学（第一性原理） | 新闻速度/加速度 → 过零点反转信号 |
| D 套息脆弱性 | 沙堆模型（演绎） | 脆弱性指数 → 四阶段状态机 → 雪崩跟随 |
| E 央行语义漂移 | NLP + 央行沟通学（归纳） | LLM 对比前后声明 → 语义距离 → 底仓偏见 |

## 快速开始

### 环境安装

```bash
pip install anthropic pandas numpy scipy
```

### 运行演示

```bash
export ANTHROPIC_API_KEY=sk-ant-xxxxx

# 策略一：NSIF 舆情驱动
python 02_NSIF舆情驱动策略_代码.py

# 策略二：宏观情景多因子
python 03_宏观情景多因子策略_代码.py

# 策略三：五策略动物园
python 04_第一性原理策略集_代码.py
```

### 推荐组合使用

```
底仓层（周度）:  策略 E 央行语义漂移 → 确定方向偏见
战略层（日内）:  策略 03 状态机 + 策略 B/D 相对价值与脆弱性
战术层（分钟）:  策略 02 NSIF + 策略 A/C 事件回归与新闻物理
```

## 关键参数速查

| 参数 | 默认值 | 所属策略 | 说明 |
|------|--------|---------|------|
| `nsif_entry_threshold` | 0.35 | 02 NSIF | NSIF 开仓阈值 |
| `nsif_strong_threshold` | 0.65 | 02 NSIF | 强信号阈值 |
| `FRAGILITY_THRESHOLD` | 0.65 | 04-D 套息 | 脆弱性临界态阈值 |
| `SHOCK_SIGMA` | 3.0 | 04-A 回归 | 冲击检测 σ 倍数 |
| `DRIFT_MAGNITUDE_THRESHOLD` | 0.6 | 04-E 语义 | 语义漂移幅度阈值 |
| `geo_risk_level` | 1 | 通用 | 0=正常, 1=升级, 2=危机 |
| `max_position_pct` | 30% | 通用 | 单资产最大仓位 |
| `daily_loss_limit` | 2% | 通用 | 日亏损熔断 |
