## 策略 A：过度反应-回归（Overreaction-Reversion）

### 假设

**第一性原理**：人类对突发威胁的反应遵循"战或逃"本能——先剧烈反应，再理性评估。金融市场是人类集体行为的映射，因此：

> **核心假设**：地缘政治冲击事件在前 30-90 分钟内造成价格过度偏离（overshoot），随后 1-4 小时内回归均值 30-60%。

**可证伪条件**：如果事件后的价格持续单边运动而不回调（如 2022 年 2 月俄乌战争爆发后的原油），则假设不成立。

### 归纳证据

| 事件 | 初始冲击 | 30min 后回调幅度 |
|------|---------|----------------|
| 2020-01 美伊苏莱曼尼事件 | XAU +2.8% (30min) | 回调 45%（1h 后涨幅收窄至 +1.5%） |
| 2024-04 以色列打击伊朗 | XAU +1.5% (15min) | 回调 60%（2h 后涨幅收窄至 +0.6%） |
| 2024-10 伊朗导弹反击以色列 | Oil +5.2% (1h) | 回调 70%（24h 后涨幅收窄至 +1.5%） |
| 2024-08 BoJ 加息冲击 | USD/JPY -3.5% (day 1) | 无显著回调（结构性转变）→ **反例** |

**归纳结论**：一次性军事打击/外交冲击大概率回调；结构性变量变化（央行政策转向）不回调。

### 关键区分：LLM 判断"一次性冲击"vs"结构性转变"

```
Prompt 核心问题：
"这条新闻描述的事件是一次性冲击（单次军事打击、外交事件、
  数据意外）还是结构性变量变化（政策转向、制度变化、联盟重组）？"

返回：
  event_type: "one_shot" | "structural"
  reversion_probability: 0.0-1.0
  expected_reversion_pct: 0.0-1.0（预期回调幅度占初始冲击的比例）
```

### 交易规则

```
1. 检测到冲击事件（任一资产 5min 变动 > 历史 3σ）

2. LLM 分类：
   - one_shot + reversion_prob > 0.6 → 启动回归交易
   - structural → 不交易（或跟随趋势）

3. 等待"峰值确认"：
   - 冲击后连续 3 根 5min K 线不再创新高/新低
   - 或冲击后 30 分钟

4. 逆向入场：
   - XAU 冲击涨 2% → 做空（目标：回调至涨 1%）
   - USD/JPY 冲击跌 1% → 做多（目标：回调至跌 0.5%）

5. 止损：
   - 冲击极值 + 0.5%（即如果冲击继续扩大则认错）

6. 最长持仓 3 小时
```

## 策略 B：避险资产相对价值（Safe-Haven Relative Value）

### 假设

**演绎起点**：三大避险资产（黄金/瑞郎/日元）的避险属性来源不同——

| 避险资产 | 避险来源 | 脆弱性 |
|---------|---------|--------|
| XAU | 零信用风险、央行储备资产、通胀对冲 | 实际利率急升时下跌 |
| CHF | 瑞士中立国地位、金融中心、低通胀 | SNB 干预汇率时走弱 |
| JPY | 日本海外资产回流、低利率融资货币 | BoJ 鸽派时避险属性削弱 |

> **核心假设**：在不同类型的危机中，三者的相对强弱不同。交易"谁比谁更强"（相对价值）比押注"谁会涨"（方向性）更稳定。

**演绎推论**：
- 美伊军事冲突 → 黄金最强（实物避险 + 央行购金）；日元其次（如果 BoJ 不干预）；瑞郎第三
- BoJ 鹰派冲击 → 日元最强（利差驱动）；黄金中性；瑞郎中性
- 美国财政危机 → 黄金最强（美元替代）；瑞郎第二；日元第三
- 全球金融系统风险 → 美元最强（Triffin 悖论下的终极避险）；三者全涨但幅度小于 USD

### 交易结构

**不做方向性交易，只做配对交易（Pair Trade）**：

```
信号：LLM 判断当前危机类型 → 确定避险排序 → 做多最强 / 做空最弱

举例（美伊军事冲突）：
  排序：XAU > JPY > CHF
  交易：做多 XAU/CHF 合成（即同时做多黄金+做空瑞郎）
  或：做多 XAU/JPY 合成（以黄金计价的日元）

优势：
  即使判断错"地缘危机是否会升级"，只要排序正确就能盈利
  比如黄金涨 2%、瑞郎涨 1% → 净赚 1%
  或黄金跌 1%、瑞郎跌 2% → 净赚 1%

风险：
  排序判错（如黄金涨但 CHF 涨更多）→ 亏损
```

### 合成方式

```
XAU/CHF 合成 = 做多 XAU/USD + 做多 USD/CHF（= 做空 CHF）
XAU/JPY 合成 = 做多 XAU/USD + 做多 USD/JPY 做空方（= 做空 JPY）

注意：上述等式在实际操作中需要按名义值配平
```

## 策略 C：新闻流物理学（News Flow Physics）

### 假设

**第一性原理**：将新闻流视为一个物理系统。单条新闻的"情感方向"不重要，重要的是新闻流的**宏观统计特性**：

| 物理量 | 新闻流对应 | 市场含义 |
|--------|----------|---------|
| 位移 x(t) | 累计新闻数量 | 市场关注度水平 |
| 速度 v(t) = dx/dt | 新闻发布频率 | 市场紧张程度 |
| 加速度 a(t) = dv/dt | 频率变化率 | 市场状态转换信号 |
| 质量 m | 新闻来源权重 | 信息可信度 |
| 力 F = ma | 加权加速度 | 实际市场冲击力 |

> **核心假设**：
> - 新闻加速度 a(t) > 0 且加速（越来越多新闻）→ 波动率即将飙升，趋势将强化
> - 新闻加速度 a(t) < 0 且减速（新闻开始减少）→ 波动率即将回落，均值回归机会
> - 新闻速度 v(t) 的极值（速度峰值）对应价格运动的加速阶段

**关键洞察**：当新闻频率从"加速"变为"减速"（加速度过零点）——这是价格拐点的领先指标。

### 计算方法

```python
# 滑动窗口计算新闻流的速度和加速度

def news_velocity(news_timestamps, window_minutes=10):
    """
    计算当前时刻的新闻流速度（条/分钟）
    """
    now = current_time()
    recent = [t for t in news_timestamps if (now - t).minutes <= window_minutes]
    return len(recent) / window_minutes

def news_acceleration(velocities):
    """
    速度的一阶差分 = 加速度
    velocities: 每分钟采样一次的速度序列
    """
    if len(velocities) < 2:
        return 0
    return velocities[-1] - velocities[-2]
```

### 交易规则

```
信号 1：加速突破（趋势追随）
  条件：v(t) > v_avg × 2.5  且  a(t) > 0（加速中）
  操作：顺趋势入场（黄金涨就多，跌就空）
  逻辑：新闻还在加速涌入，市场恐慌未到顶峰

信号 2：加速度过零（反转信号）
  条件：a(t) 从正转负（新闻开始减速）且 v(t) 仍 > v_avg
  操作：开始减仓或建立反向小仓位
  逻辑：恐慌高峰已过，信息冲击力在衰减

信号 3：速度归零（趋势结束）
  条件：v(t) 回落到 v_avg 以下
  操作：全部平仓
  逻辑：市场注意力转移，驱动力消失
```

## 策略 D：套息交易脆弱性（Carry Trade Fragility）

### 假设

**演绎起点（沙堆模型）**：

物理学中的沙堆模型告诉我们：一个系统在外力下缓慢积累应力，直到一个临界点，一粒沙子的加入导致雪崩。

日元套息交易（Carry Trade）完美符合这个模型：
- **积累阶段**：低利率环境下，全球投资者借日元→买高息资产，头寸持续积累
- **临界态**：头寸达到极端水平，市场流动性看似充裕但实际脆弱
- **触发**：一个小冲击（BoJ 加息暗示、VIX 跳升、地缘事件）→ 连锁平仓 → 雪崩

> **核心假设**：日元套息平仓不是线性的（跌 1% → 再跌 1%），而是**非线性自我强化**的（跌 1% → 触发止损 → 跌 3% → 更多止损 → 跌 5%）。识别"临界态"是策略核心。

### 临界态指标

```
Carry_Fragility_Index = w1 × CFTC_JPY_short_extreme
                      + w2 × USD_JPY_vol_term_inversion
                      + w3 × cross_asset_correlation_spike
                      + w4 × BoJ_hawkish_signal

各项含义：
  CFTC_JPY_short_extreme: 投机性日元空头持仓处于历史极值（>90th percentile）
  USD_JPY_vol_term_inversion: 短期隐含波动率 > 长期（期限结构倒挂 = 市场预感冲击）
  cross_asset_correlation_spike: 高息货币之间相关性突然飙升（齐涨齐跌 = 共同因子在驱动）
  BoJ_hawkish_signal: BoJ Tracker Score（来自 Factor 4）
```

### 交易规则

```
阶段 1：脆弱性积累（准备阶段）
  Carry_Fragility_Index > 阈值
  操作：建立小额 USD/JPY 空头底仓（5%）
  逻辑：系统处于临界态，随时可能崩溃

阶段 2：触发事件检测
  条件：USD/JPY 5min 跌 > 0.5% 且 VIX 涨 > 10%
  操作：加仓至 25%（雪崩开始）
  逻辑：套息平仓链启动，自我强化阶段

阶段 3：雪崩加速
  条件：USD/JPY 持续创新低，每 5min 跌幅加速
  操作：追踪止损跟随，不设固定止盈
  逻辑：让利润跑，非线性崩溃阶段利润可观

阶段 4：雪崩结束
  条件：USD/JPY 跌幅收窄 + VIX 开始回落
  操作：平仓 80%，保留 20% 底仓
  逻辑：主要平仓压力已释放，但尾部效应可能持续数天

风险：BoJ 紧急声明稳定汇率 → 反弹 → 止损出场
```

## 策略 E：央行语义漂移（Central Bank Semantic Drift）

### 假设

**归纳起点**：央行不会直接说"我们要加息/降息"，而是通过措辞的微妙变化来引导市场预期。这些变化往往在正式行动前 1-3 个月就开始出现。

| 措辞变化 | 含义 | 历史案例 |
|---------|------|---------|
| "patient" → "data-dependent" | 从等待变为准备行动 | Fed 2023 Q4 |
| "some time" → "coming months" | 时间框架收窄 | ECB 2024 |
| "monitoring" → "closely watching" | 关注度升级 | BoJ 2024 |
| "appropriate" → "necessary" | 行动必要性增强 | BoE 2024 |

> **核心假设**：LLM 可以检测到央行措辞的语义漂移（同一主题不同时间的措辞差异），这种漂移在市场完全消化前就已经开始——是一个有信息含量的领先信号。

### 实现方法

```
1. 建立央行声明语料库（每次会议纪要/声明/发言）

2. 对同一主题（如"通胀"、"就业"、"政策路径"）提取关键句子

3. 用 LLM 对比前后两次声明的语义距离：
   Prompt: "对比以下两段关于 [通胀] 的央行表述，
            语气偏鹰还是偏鸽了？变化幅度？"
   Output: drift_direction(+1鹰/-1鸽), drift_magnitude(0-1)

4. 当 drift_magnitude > 0.6 且方向一致连续 2 次 → 生成交易信号

5. 信号到资产的映射：
   Fed 鹰派漂移 → 做多 USD, 做空 XAU
   BoJ 鹰派漂移 → 做空 USD/JPY
   ECB 鸽派漂移 → 做空 EUR/USD
```

**信号频率**：低频（每次央行事件后更新，月均 4-8 次信号）

**与高频策略的结合**：语义漂移信号作为"底仓方向偏见"，高频策略在此基础上做日内交易。

## 策略组合建议

单一策略容易在某种市场状态下失效。推荐组合使用：

```
底仓层（长期偏见，周度调整）：
  └── 策略 E：央行语义漂移 → 确定底仓方向

战略层（日内方向，4小时调整）：
  ├── 策略 B：避险资产相对价值 → 确定配对交易方向
  └── 策略 D：套息脆弱性 → 监控崩溃触发器

战术层（分钟级交易）：
  ├── 策略 A：过度反应回归 → 事件后捕捉回调
  └── 策略 C：新闻流物理学 → 波动率状态切换

资金分配：
  底仓层 20% | 战略层 40% | 战术层 40%
```
