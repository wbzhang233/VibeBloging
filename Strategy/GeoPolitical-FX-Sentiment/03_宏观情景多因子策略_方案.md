# 宏观情景驱动多因子高频策略
## 美伊战争 · 2026 年 4 月实战方案

> **与 02 方案的根本区别**：02 方案以 NSIF 单一情感因子为核心，是通用舆情框架的工程化落地。本方案**直接锚定当前宏观情景**——美伊军事对峙、BoJ 正常化、特朗普关税滞胀——构建多因子信号矩阵，策略逻辑与当下市场结构深度绑定。
>
> **定位**：商业银行资金营运中心，外汇 + 贵金属交易台，日内高频（5 分钟 K 线级别）

---

## 一、当前市场的三个核心矛盾

在设计策略之前，必须先厘清当前市场的核心驱动力——不是泛泛的"地缘政治风险"，而是三个具体的、互相交织的矛盾：

### 矛盾 1：避险需求 vs. 美元信用动摇

**传统剧本**：地缘危机 → 买美元（避险）+ 买黄金（避险）

**当前现实**：美元的避险地位正被侵蚀——
- 特朗普关税政策动摇全球对美国制度稳定性的信心
- 美国财政赤字持续扩大，国债可持续性被质疑
- 全球央行持续减持美债、增持黄金（年购 1,000+ 吨）
- 结果：地缘危机中黄金涨幅远超美元，甚至出现"美元与黄金同方向"的异常

**策略含义**：做多黄金的信心远高于做多美元。在极端避险事件中，XAU/USD 是最确定的多头标的。

### 矛盾 2：BoJ 正常化 vs. 全球套息交易

**核心变量**：日本央行历史性加息周期（从负利率到约 0.75-1.25%）

**传导链条**：
```
BoJ 加息 → 日美利差收窄 → 日元套息交易(Carry Trade)平仓
  → USD/JPY 急跌（日元升值）
  → 全球风险资产承压（套息资金回流日本）
  → VIX 跳升
```

**2024 年 8 月前车之鉴**：BoJ 加息 25bp → USD/JPY 从 161 暴跌至 141 → VIX 日内触 65

**策略含义**：USD/JPY 是本轮最大的趋势性空头机会。每次 BoJ 相关新闻（加息暗示、通胀数据超预期）都是做空 USD/JPY 的触发器。

### 矛盾 3：能源供给恐慌 vs. 需求侧疲软

**供给侧**：霍尔木兹海峡封锁风险 → 油价跳涨 → 输入性通胀
**需求侧**：全球贸易战 + 高利率 → 经济放缓 → 需求下行

**实际表现**：油价在 $70-90 区间剧烈震荡，方向不明——因为多空力量势均力敌。

**策略含义**：原油不适合做方向性策略，但油价波动是其他资产的**领先指标**——油价急涨时做多 USD/CAD（加拿大石油出口国受益）、做空 EUR/USD（欧洲能源进口依赖）。

## 二、四因子信号矩阵

不同于 NSIF 的"一个公式算一切"，本策略使用四个独立因子，各自捕捉不同的市场驱动力：

### Factor 1：新闻冲击强度（News Shock Intensity, NSI）

**核心思路**：不关心新闻"说了什么"（情感方向），只关心新闻"有多大动静"（冲击强度）。

**计算方法**：
```
NSI(t) = count(news in [t-30min, t]) × avg_urgency_score

其中 urgency_score:
  breaking/flash = 3.0
  headline       = 2.0
  analysis       = 1.0
  commentary     = 0.5
```

**使用方式**：NSI 是波动率的领先指标——NSI 飙升时，接下来 5-30 分钟的市场波动率大概率放大。

- NSI > 阈值 → 进入"高波动模式"：买入跨式期权（straddle）或扩大止损宽度
- NSI 持续低位 → "静默模式"：缩小仓位、收紧止损

### Factor 2：跨资产领先-滞后信号（Cross-Asset Lead-Lag）

**核心发现**：在地缘政治事件中，资产反应速度有先后顺序——

```
信息传导链（快 → 慢）：
  原油期货（反应最快，1-2分钟）
    → 黄金（2-5分钟）
      → 日元/瑞郎（3-8分钟）
        → 欧元/英镑（5-15分钟）
          → 股指期货（5-20分钟）
```

**策略逻辑**：用"快"资产的异动预测"慢"资产的走势。

具体信号规则：

| 领先资产异动 | 预测 | 滞后资产交易 | 延迟窗口 |
|-------------|------|------------|---------|
| Brent 5min 涨 > 1.5% | 能源冲击升级 | 做多 XAU/USD, 做多 USD/CAD | 2-8 分钟 |
| Brent 5min 跌 > 1.5% | 需求恐慌 | 做空 AUD/USD, 做空 USD/CAD | 2-8 分钟 |
| XAU 5min 涨 > 0.8% | 避险升温 | 做空 USD/JPY, 做空 USD/CHF | 3-10 分钟 |
| USD/JPY 5min 跌 > 0.5% | 套息平仓启动 | 做空 AUD/USD, 做多 XAU/USD | 3-10 分钟 |
| VIX 日内涨 > 15% | 恐慌蔓延 | 做多 XAU/USD, 做空 EUR/USD | 5-15 分钟 |

### Factor 3：宏观情景状态机（Macro Regime State Machine）

**核心思路**：市场不是"一直在避险"或"一直在冒险"，而是在几种明确的状态之间切换。不同状态下应该用完全不同的交易逻辑。

**四种状态定义**：

```
┌──────────────────────────────────────────────────┐
│  状态 A：地缘紧张升级（Escalation）              │
│  特征：军事冲突新闻、制裁升级、霍尔木兹威胁      │
│  持续：数小时到数天                               │
│  信号：原油急涨、黄金急涨、VIX 跳升              │
│  策略：积极做多黄金，做空风险货币                  │
├──────────────────────────────────────────────────┤
│  状态 B：恐慌消化（Digestion）                    │
│  特征：冲击事件后 2-8 小时，市场寻找新均衡        │
│  持续：数小时                                     │
│  信号：波动率维高但方向不明，成交量下降            │
│  策略：缩小仓位，等待方向确认                      │
├──────────────────────────────────────────────────┤
│  状态 C：地缘缓和（De-escalation）                │
│  特征：外交接触、停火声明、谈判传闻               │
│  持续：数小时到数天                               │
│  信号：原油回落、黄金回落、风险货币反弹            │
│  策略：黄金短空（回调交易），做多欧元/澳元         │
├──────────────────────────────────────────────────┤
│  状态 D：宏观数据驱动（Data-Driven）              │
│  特征：地缘风险暂退，CPI/非农/央行会议主导        │
│  持续：数据窗口期（发布前后各 2 小时）             │
│  信号：波动率回归正常，利差主导汇率                │
│  策略：数据驱动的标准宏观交易（看利差、看预期差）  │
└──────────────────────────────────────────────────┘
```

**状态切换逻辑**（LLM 辅助判断）：

```python
def detect_regime(recent_news, price_action, current_regime):
    """
    输入最近 1 小时的新闻流 + 价格行动，判断当前宏观情景状态。
    使用 LLM 做分类判断（非传统规则引擎）。
    """
    prompt = f"""
    当前状态: {current_regime}
    最近1小时新闻摘要: {recent_news_summary}
    价格行动: 原油{oil_chg}%, 黄金{gold_chg}%, VIX={vix_level}

    判断当前应处于哪种状态:
    A: 地缘紧张升级（新的军事/制裁事件正在发生）
    B: 恐慌消化（重大事件已发生，市场在消化中）
    C: 地缘缓和（出现外交/停火信号）
    D: 宏观数据驱动（地缘暂退，数据/央行主导）

    返回: 状态字母 + 置信度(0-1) + 一句话理由
    """

```

### Factor 4：BoJ 政策信号追踪器（BoJ Signal Tracker）

**为什么单独做一个因子**：BoJ 正常化是当前 USD/JPY 最大的结构性驱动力，值得专门追踪。

**监控信号源**：
1. BoJ 政策声明、会议纪要
2. 日本 CPI / 工资数据
3. BoJ 官员发言（植田和男、副行长等）
4. 日本国债收益率（JGB 10Y）异动
5. 日元隔夜掉期利率

**信号量化**：
```
BoJ_Score(t) = Σ signal_weight × signal_direction × recency_decay

signal_direction:
  +1 = 鹰派（暗示加息、通胀超预期、工资上涨）
  -1 = 鸽派（暂停加息、经济放缓、日元过度升值担忧）

signal_weight:
  正式政策声明 = 5.0
  官员公开发言 = 3.0
  经济数据     = 2.0
  市场隐含     = 1.0

recency_decay:
  24小时内 = 1.0
  1-3天    = 0.6
  3-7天    = 0.3
  >7天     = 0.1
```

**交易规则**：
- BoJ_Score > +3.0 → 做空 USD/JPY（预期 BoJ 加息 → 日元升值）
- BoJ_Score < -2.0 → 暂停 USD/JPY 空头（BoJ 可能鸽派转向）
- BoJ_Score 在 -2 到 +3 之间 → 维持现有仓位，不新增

## 三、资产组合与具体交易规则

### 3.1 核心交易对（按确定性排序）

| 优先级 | 资产 | 方向偏见 | 核心逻辑 | 信号来源 |
|--------|------|---------|---------|---------|
| **1** | XAU/USD | 结构性多头 | 央行购金+地缘溢价+美元信用动摇 | Factor1(NSI) + Factor3(Regime A) |
| **2** | USD/JPY | 结构性空头 | BoJ 正常化驱动日美利差收窄 | Factor4(BoJ Tracker) + Factor2(Lead-Lag) |
| **3** | USD/CHF | 情景性空头 | 避险事件中瑞郎受益 | Factor3(Regime A) |
| **4** | USD/CAD | 油价联动 | 加拿大石油出口国，油涨则 CAD 强 | Factor2(Oil Lead-Lag) |
| **5** | EUR/USD | 情景性空头 | 欧洲能源依赖、ECB 偏鸽 | Factor3(Regime A/D) |
| **6** | AUD/USD | 风险情绪指标 | 典型风险货币，恐慌中被抛售 | Factor2(Lead-Lag) + Factor3(Regime) |

### 3.2 状态-资产交易矩阵

**每种宏观状态下的具体操作**：

#### 状态 A：地缘紧张升级

```
触发条件：
  - LLM Regime = A, 置信度 > 0.7
  - 或：Brent 5min > +2% 且 XAU 5min > +0.5% 同时发生

操作：
  XAU/USD  →  加仓多头（目标仓位 30%，ATR×2 止损）
  USD/JPY  →  加仓空头（目标仓位 25%，ATR×2 止损）
  USD/CHF  →  开仓空头（目标仓位 15%，ATR×1.5 止损）
  EUR/USD  →  开仓空头（目标仓位 10%，ATR×1.5 止损）
  AUD/USD  →  开仓空头（目标仓位 10%，ATR×1.5 止损）

止盈规则：
  - XAU/USD: 不设固定止盈，追踪止损（回撤 0.8% 平仓）
  - 其他货币对: 2×ATR 止盈

最长持仓: 6 小时（地缘事件衰减较快）
```

#### 状态 B：恐慌消化

```
触发条件：
  - 状态 A 已持续 > 2 小时
  - 且：NSI 从峰值回落 > 50%
  - 且：价格波动放缓（5min ATR < 前期 ATR 的 60%）

操作：
  所有仓位减半
  不开新仓
  收紧止损至 ATR×1.0

持续监控：
  - 若 NSI 重新飙升 → 回到状态 A
  - 若 60 分钟内无新高危新闻 → 转入状态 C 或 D
```

#### 状态 C：地缘缓和

```
触发条件：
  - LLM Regime = C, 置信度 > 0.65
  - 或：连续 2 小时无地缘升级新闻，且风险资产反弹

操作：
  XAU/USD  →  减仓至 15%（保留部分多头底仓）
  USD/JPY  →  维持空头（BoJ 逻辑不因地缘缓和而改变）
  EUR/USD  →  翻多（15%仓位，ATR×1.5 止损，目标 2×ATR）
  AUD/USD  →  翻多（10%仓位，ATR×1.5 止损，目标 2×ATR）
  USD/CHF  →  平仓

特别注意：
  地缘缓和往往是假象——保留 XAU 底仓作为"保险"
```

#### 状态 D：宏观数据驱动

```
触发条件：
  - 地缘风险暂退
  - 重要数据发布前后 2 小时窗口（CPI/非农/央行决议）

操作：
  根据数据预期差交易：
  - CPI > 预期 → 做空 XAU（实际利率上升预期）+ 做多 USD
  - CPI < 预期 → 做多 XAU + 做空 USD
  - 非农 > 预期 → 做多 USD/JPY（暂时）+ 做空 XAU（暂时）
  - BoJ 加息 → 做空 USD/JPY（Factor 4 优先）
  - Fed 鹰派意外 → 做多 USD + 做空 XAU

仓位较小（每笔 10%），止损紧（ATR×1.0），持仓短（< 2 小时）
```

### 3.3 仓位管理（与 02 方案不同的方法）

02 方案用 Kelly + 波动率调整。本方案使用**情景加权风险预算法**：

```
每日总风险预算 = 账户净值 × 2%（最大日亏损容忍度）

各状态的风险预算分配：
  状态 A（升级）: 可用 70% 预算（高确定性方向）
  状态 B（消化）: 可用 20% 预算（降低敞口）
  状态 C（缓和）: 可用 50% 预算（方向性反转交易）
  状态 D（数据）: 可用 40% 预算（标准宏观交易）

单笔风险 = 入场价到止损价的距离 × 仓位量
仓位量 = 风险预算 / 单笔风险距离（ATR-based）
```

**举例**：
- 账户净值 1 亿人民币，日风险预算 = 200 万
- 状态 A 下，可用预算 = 140 万
- XAU/USD 入场 3,200，止损 3,168（ATR×2 = 32 点），每点价值 100 美元
- 单笔风险 = 32 × 100 = $3,200/手
- 可开仓量 = 140 万 ÷ 7.2（汇率） ÷ 3,200 ≈ 60 手

## 四、LLM 分析引擎设计（与 02 方案的差异）

02 方案让 LLM 逐条新闻分析出 NSIF 分量（direction/intensity/confidence）。

本方案让 LLM 做**更高层次的判断**——不分析单条新闻的情感，而是综合最近一批新闻做"情景研判"。

### 4.1 情景研判 Prompt

```
你是一位商业银行资金营运中心的高级宏观策略师。

以下是最近 1 小时的路透社新闻标题（按时间倒序）：
{news_headlines}

当前市场行情：
  XAU/USD: {gold_price} (1h chg: {gold_chg}%)
  Brent: {oil_price} (1h chg: {oil_chg}%)
  USD/JPY: {usdjpy} (1h chg: {usdjpy_chg}%)
  DXY: {dxy} (1h chg: {dxy_chg}%)
  VIX: {vix}

请回答以下问题（JSON 格式）：

1. regime: 当前市场处于哪种状态？
   A=地缘紧张升级 / B=恐慌消化 / C=地缘缓和 / D=宏观数据驱动

2. regime_confidence: 状态判断的置信度 (0.0-1.0)

3. key_risk: 未来 4 小时最大的单一风险事件是什么？

4. gold_bias: 黄金未来 2 小时的方向偏见
   strong_bull / bull / neutral / bear / strong_bear

5. jpy_bias: 日元未来 2 小时的方向偏见（注意 USD/JPY 下跌=日元走强）
   strong_bull / bull / neutral / bear / strong_bear

6. oil_signal: 原油价格变动对 FX 的传导判断
   risk_off_spreading / supply_shock / demand_fear / neutral

7. actionable_pairs: 当前最值得交易的 2-3 个货币对及方向
   格式: [{"pair": "XAU/USD", "direction": "long", "conviction": 0.85}, ...]

8. avoid_pairs: 当前应该回避的货币对及原因

9. reasoning: 50 字以内的核心逻辑
```

### 4.2 调用频率与成本控制

| 触发条件 | 调用 | 模型 | 估计成本 |
|---------|------|------|---------|
| 常规：每 15 分钟 | 情景研判 | Claude Haiku | ~$0.002/次 |
| NSI 跳升（新闻潮涌） | 紧急研判 | Claude Haiku | ~$0.003/次 |
| BoJ/Fed/ECB 声明 | 深度分析 | Claude Sonnet | ~$0.02/次 |
| 日常开销 | 每日约 100-150 次 | 混合 | ~$0.50-1.00/天 |

**月成本估算**：$15-30/月（相对于交易台的收益规模，成本可忽略）

## 五、具体交易机会清单（基于 2026 年 4 月情景）

以下是当前宏观环境下最具体、最可行动的交易机会：

### 机会 1：XAU/USD 趋势多头 + 事件加速

```
逻辑：
  结构性牛市（央行购金 + 美元信用动摇 + 地缘溢价）
  + 事件驱动加速器（美伊升级 → 避险买盘涌入）

入场方式：
  - 底仓：任何回调至 20 日均线附近加仓（趋势跟随）
  - 事件仓：状态 A 触发时立即加仓（事件驱动）

仓位：底仓 15% 常驻 + 事件仓最高加至 30%
止损：底仓 ATR(20)×3（宽止损保趋势）；事件仓 ATR(5)×2
目标：不设固定止盈，追踪止损退出

预期胜率：60-65%（趋势方向正确时单笔盈亏比可达 3:1+）
```

### 机会 2：USD/JPY 结构性空头

```
逻辑：
  BoJ 正常化周期 → 日美利差从 350bp+ 持续收窄
  每次 BoJ 鹰派信号 → USD/JPY 急跌 1-3%
  2024年8月先例：20天内跌 12%

入场方式：
  - 底仓：BoJ_Score > +3 时建仓或加仓
  - 事件仓：BoJ 正式加息声明后 → 追空
  - 技术仓：USD/JPY 反弹至 20日均线上方 → 逆势空

仓位：底仓 15% + 事件仓最高 25%
止损：每次入场 ATR(20)×2（约 200-300 pips）
目标：趋势跟随，追踪止损退出

关键风险：Fed 意外超鹰 → USD/JPY 反弹；需在 Fed 声明前减仓
```

### 机会 3：油价跳涨 → USD/CAD 做多联动

```
逻辑：
  加拿大是 G7 中唯一的净石油出口国
  Brent 每涨 $10 → USD/CAD 倾向下跌 200-400 pips（CAD 升值）
  但 USD/CAD 的反应通常滞后原油 5-15 分钟

入场方式：
  - Brent 5min 涨 > 1.5% 时 → 做空 USD/CAD（即做多 CAD）
```

## 六、风控体系

### 6.1 三层风控架构

```
第一层：单笔风控
  - 每笔交易止损不超过日风险预算的 30%
  - 入场前必须确定止损价位（无止损不下单）

第二层：组合风控
  - 总敞口不超过净值的 80%
  - 高相关性资产（如 XAU + CHF + JPY 全是避险）合并计算敞口
  - 避险类总敞口 ≤ 50%，风险类总敞口 ≤ 30%

第三层：日/周风控
  - 日亏损达 2% → 全部平仓，当日暂停
  - 周亏损达 3.5% → 全部平仓，本周暂停
  - 连续亏损 3 笔 → 仓位减半，直到盈利 1 笔恢复
```

## 八、v2.0 市场机制修订：滞胀机制 + 第五因子 RHFI（2026-04-13）

> **修订触发**：2026-04-13 实时市场数据显示——VIX +10.19%，XAU/USD -0.91%（$4,744），CL=F +6.30%，XOM -1.63%。这组数据清晰揭示：**当前市场处于"滞胀感知"机制**，而非 v1.0 中"情景 A 升级态"的避险逻辑。黄金在 VIX 急涨时反而下跌，是因为油价飙升→通胀→加息预期的叙事压倒了地缘政治避险买盘。

### 8.1 状态机 v2.0：新增状态 E（滞胀态）

在原有四状态（A 升级、B 消化、C 缓和、D 数据驱动）的基础上，**新增状态 E**：

| 状态 | 名称 | 触发条件 | 主导叙事 | 黄金方向 | 原油方向 | USD 方向 |
|------|------|---------|---------|---------|---------|---------|
| A | 升级恐慌 | 军事冲突、核威胁急剧升级 | 避险 | ↑↑ | ↑↑ | ±（分裂）|
| B | 消化期 | 冲突持续但无新突破 | 观望 | ↑（缓）| ↑（缓）| ± |
| C | 缓和期 | 停火信号、外交突破 | 缓和 | ↓↓ | ↓↓ | + |
| D | 数据驱动 | 无重大地缘事件，经济数据主导 | 宏观基本面 | 视利率 | 视需求 | 视数据 |
| **E** | **滞胀态** | **供应中断持续 → 油价高位 → 通胀预期升温 → 加息担忧** | **通胀/利率** | **↓（加息压制）** | **↑↑** | **+（Fed鹰派）** |

4. TIPS 盈亏平衡通胀率快速上行（需实时数据确认）

**当前状态**：**状态 E（滞胀态）已激活** — 2026-04-13

### 8.2 第五因子：RHFI（加息恐惧指数）

在原有四因子（NSI、跨资产领先滞后、状态机、BoJ Tracker）之上，新增**第五因子**：

```
因子五：RHFI（Rate Hike Fear Index，加息恐惧指数）

构建逻辑：
  RHFI_t = Σ (news_i.rate_hike_signal × w_source × λ_rate)

其中：
  rate_hike_signal = LLM 提取：
    - 正向信号（加息方向）：rate hike, CPI surprise, hawkish,
                              inflation spike, oil-driven inflation,
                              Fed staying higher, tightening
    - 负向信号（降息方向）：rate cut, dovish, recession, pivot,
                              demand destruction, growth slowdown

  λ_rate 衰减半衰期 = 4h（利率预期变化比地缘政治预期更快）

  RHFI 范围：[-1, +1]
  正值 = 加息恐惧（压制黄金）
  负值 = 降息预期（支撑黄金）
```

**五因子联合信号矩阵（状态 E 下）**：

```python
# 状态 E 专属信号合成规则
if regime_state == "E":  # 滞胀态
    # 黄金信号：RHFI 主导，NSI 地缘因子大幅折扣
    gold_signal = -1.5 * RHFI + 0.3 * NSI_geopolitical

    # 原油信号：NSI + Lead-Lag 放大
    oil_signal = 1.8 * NSI_supply + 1.2 * lead_lag_oil

    # CAD 信号：油价联动放大
    cad_signal = 1.6 * oil_signal * oil_cad_correlation

    # JPY 信号：滞胀下日元矛盾加剧（能源进口负 vs 避险正）
    jpy_signal = 0.4 * boj_tracker - 0.6 * RHFI * japan_energy_import_sensitivity

    # CHF 信号：与黄金解耦，独立避险（美元信用危机驱动）
    chf_signal = 0.8 * dollar_credibility_crisis_index  # 独立因子

else:  # 原有 A/B/C/D 状态逻辑不变
    gold_signal = geo_amplifier * NSIF_gold  # 原逻辑
```

### 8.3 跨资产领先滞后序列：状态 E 修订版

**原始序列（v1.0，适用于 A/B/C 状态）**：
```
地缘冲击 → 黄金（最快）→ JPY/CHF → EUR/GBP → 股票
```

**状态 E 修订序列（通胀机制主导）**：
```
供应冲击 → 原油（最快，T+0）
              ↓
         国债抛售（收益率上行，T+5~15分钟）
              ↓
         黄金下跌（加息预期压制，T+15~45分钟）
              ↓
         CAD 上涨（商品货币受益，T+30~90分钟）
              ↓
         EUR/JPY 下跌（能源进口成本上升，T+60~240分钟）
              ↓
         股票下跌（滞胀定价，T+2~8小时）
```

**关键变化**：
- 黄金不再是第一个上涨的资产，反而是第一个**下跌**的资产（在利率预期叙事主导下）
- CAD 从"风险货币"变为**状态 E 的主要多头**
- EUR/JPY 的下跌来自能源进口成本，而非传统"避险"逻辑

### 8.5 状态 E 的退出条件与机制切换交易

**退出状态 E，进入其他状态的信号**：

```

  触发条件：伊朗攻击海湾港口/核武器威胁/以色列城市直接受威胁
  交易响应：立即平空黄金，反手做多，同时做多 CHF

  触发条件：停火协议签署 + 霍尔木兹开放确认 + 油价暴跌
```

**机制切换是最重要的 Alpha 窗口**：当 RHFI 从高位（加息恐惧）快速下降至低位时，黄金将经历从"被压制"到"解放"的机制性修复行情——这种行情通常发生在停火协议后 2-6 小时，油价暴跌之后，**速度极快、幅度极大**（可能 2-4 小时内涨 3-5%）。
