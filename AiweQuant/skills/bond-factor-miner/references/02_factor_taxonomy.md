## 一、因子分类总览

债券类因子按驱动机制分为六大类，共覆盖 30+ 候选因子：

```
┌─────────────────────────────────────────────────────────────┐
│               债券因子体系（6 大类）                          │
├──────────────┬──────────────────────────────────────────────┤
│ 机构行为类   │ 直接反映市场参与者的买卖行为，先行性最强       │
│ 宏观经济类   │ 反映经济基本面对利率中枢的决定作用             │
│ 资金面类     │ 反映货币市场流动性状况对债市的传导             │
│ 技术面类     │ 基于价格/收益率数据，捕捉趋势与均值回归        │
│ 政策面类     │ 反映货币政策操作和监管政策的信号               │
│ 情绪面类     │ 反映市场整体风险偏好和交易情绪                 │
└──────────────┴──────────────────────────────────────────────┘
```

## 二、第一类：机构行为因子

> **逻辑基础**：机构净买入量是债券市场最直接的供需信号。保险/基金/券商的大规模配置行为往往领先于价格变化，因为其投资决策基于更长维度的利率判断或负债久期匹配需求。

### 2.1 净买入流量因子（Flow_Net）

| 属性 | 说明 |
|------|------|
| 中文名 | 机构净买入流量因子 |
| 英文代码 | `flow_net_{机构名}` |
| 所属数据 | 保险/基金/券商等机构超长国债净买入量（亿元） |
| 构建公式 | `Flow_Net(t) = NetBuy(t)` |
| 业务释义 | 直接反映机构当期的配置意愿。净买入量大幅正增加，表明机构看多，有助于推动收益率下行 |
| 预期方向 | **负相关**（净买入↑ → 需求增加 → 收益率↓） |

```python
def build_net_buying_factor(df: pd.DataFrame, col: str) -> pd.Series:
    """直接使用净买入量作为因子"""
    return df[col].copy().rename(f"flow_net_{col}")
```

---

### 2.2 净买入滚动均值因子（Flow_MA）

| 属性 | 说明 |
|------|------|
| 中文名 | 净买入量滚动均值因子 |
| 英文代码 | `flow_ma{W}_{机构名}` |
| 构建公式 | `Flow_MA(t, W) = Mean(NetBuy(t-W+1), ..., NetBuy(t))` |
| 参数 | W = 4（月均）/ 12（季均） |
| 业务释义 | 平滑短期波动，反映机构配置的趋势性意愿。趋势性增持更能代表机构的基本面判断 |
| 预期方向 | **负相关** |

```python
def build_flow_ma_factor(df: pd.DataFrame, col: str, window: int = 4) -> pd.Series:
    return df[col].rolling(window).mean().rename(f"flow_ma{window}_{col}")
```

---

### 2.3 净买入量变化率因子（Flow_MoM）

| 属性 | 说明 |
|------|------|
| 中文名 | 净买入量环比变化率 |
| 英文代码 | `flow_mom_{机构名}` |
| 构建公式 | `Flow_MoM(t) = (NetBuy(t) - NetBuy(t-1)) / |NetBuy(t-1)| + ε` |
| 业务释义 | 捕捉净买入量的加速或减速信号。边际变化往往是趋势转折的早期信号 |
| 预期方向 | **负相关**（净买入加速↑ → 购买力增强 → 收益率↓） |

```python
def build_flow_mom_factor(df: pd.DataFrame, col: str) -> pd.Series:
    eps = 1e-6
    mom = df[col].diff() / (df[col].shift(1).abs() + eps)
    return mom.rename(f"flow_mom_{col}")
```

---

### 2.4 累积净买入量因子（Flow_Cumsum）

| 属性 | 说明 |
|------|------|
| 中文名 | 滚动累积净买入量 |
| 英文代码 | `flow_cumsum{W}_{机构名}` |
| 构建公式 | `Flow_Cumsum(t, W) = Sum(NetBuy(t-W+1), ..., NetBuy(t))` |
| 参数 | W = 12（近3个月） |
| 业务释义 | 反映机构在一段时间内的累计配置力度，捕捉中期趋势性建仓行为 |
| 预期方向 | **负相关** |

```python
def build_flow_cumsum_factor(df: pd.DataFrame, col: str, window: int = 12) -> pd.Series:
    return df[col].rolling(window).sum().rename(f"flow_cumsum{window}_{col}")
```

---

### 2.5 多机构综合净买入因子（Flow_Composite）

| 属性 | 说明 |
|------|------|
| 中文名 | 多机构综合净买入因子 |
| 英文代码 | `flow_composite` |
| 构建公式 | `Flow_Composite(t) = w1×NetBuy_ins(t) + w2×NetBuy_fund(t) + w3×NetBuy_broker(t)`（等权或按历史IC加权） |
| 业务释义 | 综合多类机构的买卖行为，避免单一机构行为的噪声，更全面反映市场整体配置倾向 |
| 预期方向 | **负相关** |

---

### 2.6 机构行为分歧因子（Flow_Divergence）

| 属性 | 说明 |
|------|------|
| 中文名 | 机构买卖行为分歧度 |
| 英文代码 | `flow_divergence` |
| 构建公式 | `Flow_Div(t) = Std(NetBuy_ins(t), NetBuy_fund(t), NetBuy_broker(t))` |
| 业务释义 | 机构间分歧度高时，市场方向不确定性增大，往往预示收益率波动放大 |
| 预期方向 | **正相关**（分歧↑ → 不确定性↑ → 收益率波动率↑，方向不明） |

## 三、第二类：宏观经济因子

> **逻辑基础**：宏观经济基本面决定利率的长期中枢。经济强劲时，通胀预期和货币收紧预期推高收益率；经济疲弱时，收益率倾向下行。

### 3.1 CPI 同比因子

| 属性 | 说明 |
|------|------|
| 中文名 | CPI 同比变化因子 |
| 英文代码 | `macro_cpi_yoy` |
| 构建公式 | `CPI_YoY(t) = (CPI(t) - CPI(t-12)) / CPI(t-12) × 100` |
| 业务释义 | 通胀水平是名义利率的核心组成部分（费雪方程）。CPI 上行预示货币收紧压力，推高收益率 |
| 预期方向 | **正相关**（CPI↑ → 通胀预期↑ → 收益率↑） |

---

### 3.2 PMI 扩散因子

| 属性 | 说明 |
|------|------|
| 中文名 | 制造业 PMI 与荣枯线偏差 |
| 英文代码 | `macro_pmi_dev` |
| 构建公式 | `PMI_Dev(t) = PMI(t) - 50` |
| 业务释义 | PMI 高于 50 表示扩张，经济向好预期会压低债券需求、推高收益率 |
| 预期方向 | **正相关**（PMI↑ → 经济预期好转 → 收益率↑） |

---

### 3.3 社融-GDP 缺口因子

| 属性 | 说明 |
|------|------|
| 中文名 | 社融增速与名义 GDP 增速之差 |
| 英文代码 | `macro_sf_gdp_gap` |
| 构建公式 | `SF_GDP_Gap(t) = ΔSocialFinancing_YoY(t) - NominalGDP_Growth(t)` |
| 业务释义 | 衡量信用扩张相对于实体经济的超额程度。缺口为正表示流动性宽裕，有利于债市 |
| 预期方向 | **负相关**（缺口↑ → 流动性超额 → 收益率↓） |

## 四、第三类：资金面因子

> **逻辑基础**：货币市场利率（DR007 等）是债券收益率的短端锚定。资金面宽松时，机构加杠杆做多长端，压缩期限利差，推动长端收益率下行。

### 4.1 DR007 利率水平因子

| 属性 | 说明 |
|------|------|
| 中文名 | DR007 滚动均值 |
| 英文代码 | `liq_dr007_ma` |
| 构建公式 | `DR007_MA(t, W) = Mean(DR007(t-W+1), ..., DR007(t))`，W=4 |
| 业务释义 | 代表货币市场资金成本中枢。DR007 越高，债券融资成本越高，机构加杠杆意愿下降，利空长端 |
| 预期方向 | **正相关**（DR007↑ → 资金成本↑ → 收益率↑） |

---

### 4.2 期限利差因子

| 属性 | 说明 |
|------|------|
| 中文名 | 10年期与2年期国债利差 |
| 英文代码 | `liq_term_spread` |
| 构建公式 | `TermSpread(t) = Yield_10Y(t) - Yield_2Y(t)` |
| 业务释义 | 期限利差过低（曲线平坦）时，机构倾向于拉长久期捕捉超额收益，增加超长端需求，收益率趋于下行 |
| 预期方向 | **非线性**（利差过低/过高均影响30年期配置需求） |

---

### 4.3 资金利率波动率因子

| 属性 | 说明 |
|------|------|
| 中文名 | DR007 滚动波动率 |
| 英文代码 | `liq_dr007_vol` |
| 构建公式 | `DR007_Vol(t) = Std(DR007(t-W+1), ..., DR007(t))`，W=4 |
| 业务释义 | 资金面波动加大时，机构持仓不确定性上升，倾向于降低久期，利空超长端 |
| 预期方向 | **正相关**（资金波动↑ → 不确定性↑ → 收益率↑） |

## 五、第四类：技术面因子

> **逻辑基础**：价格的趋势性和均值回归特征在利率市场中同样存在，技术因子通过历史价格规律捕捉未来收益率变动。

### 5.1 动量因子

| 属性 | 说明 |
|------|------|
| 中文名 | 收益率滚动变化动量 |
| 英文代码 | `tech_momentum_{W}` |
| 构建公式 | `Momentum(t, W) = Yield(t) - Yield(t-W)` |
| 参数 | W = 4（短期）/ 12（中期） |
| 业务释义 | 利率市场存在趋势延续性。近期收益率下行趋势往往因机构追涨而延续 |
| 预期方向 | **正相关**（收益率近期下行 → 趋势延续 → 未来继续下行；负号代表正向因子） |

```python
def build_momentum_factor(yield_series: pd.Series, window: int = 4) -> pd.Series:
    return yield_series.diff(window).rename(f"tech_momentum_{window}")
```

---

### 5.2 均线偏差因子

| 属性 | 说明 |
|------|------|
| 中文名 | 收益率与长期均线的偏离度 |
| 英文代码 | `tech_ma_dev` |
| 构建公式 | `MA_Dev(t) = (Yield(t) - MA(t, W)) / MA(t, W) × 100`，W=52 |
| 业务释义 | 收益率大幅高于长期均线时（过高估值），存在均值回归的下行压力 |
| 预期方向 | **正相关**（偏差↑ → 高估 → 均值回归压力 → 收益率↓，需注意因子方向） |

```python
def build_ma_deviation_factor(yield_series: pd.Series, window: int = 52) -> pd.Series:
    ma = yield_series.rolling(window).mean()
    return ((yield_series - ma) / ma * 100).rename(f"tech_ma_dev_{window}")
```

---

### 5.3 历史分位数因子

| 属性 | 说明 |
|------|------|
| 中文名 | 收益率历史分位数 |
| 英文代码 | `tech_quantile_{W}` |
| 构建公式 | `Quantile(t, W) = Rank(Yield(t) in Yield(t-W+1,...,t)) / W × 100` |
| 参数 | W = 52（近1年分位） / 260（近5年分位） |
| 业务释义 | 分位数高表示当前收益率处于历史高位，性价比高，配置价值强，倾向回落 |
| 预期方向 | **正相关**（分位数高 → 高收益率 → 均值回归 → 收益率下行） |

## 六、第五类：政策面因子

> **逻辑基础**：货币政策直接影响资金成本和市场预期，是中短期内最重要的利率驱动因素之一。

### 6.1 MLF 利率水平因子

| 属性 | 说明 |
|------|------|
| 中文名 | MLF 1年期操作利率 |
| 英文代码 | `policy_mlf_rate` |
| 业务释义 | MLF 利率是国内无风险利率的政策锚。MLF 降息往往触发一轮债市行情 |
| 预期方向 | **正相关**（MLF 利率↑ → 政策收紧信号 → 收益率↑） |

---

### 6.2 MLF 利率变化因子

| 属性 | 说明 |
|------|------|
| 中文名 | MLF 利率环比变化 |
| 英文代码 | `policy_mlf_change` |
| 构建公式 | `MLF_Change(t) = MLF_Rate(t) - MLF_Rate(t-1)` |
| 业务释义 | 降息信号（负值）是债市最强劲的利好催化剂 |
| 预期方向 | **正相关**（MLF 降息 → 收益率下行） |

---

### 6.3 逆回购净投放因子

| 属性 | 说明 |
|------|------|
| 中文名 | 央行公开市场逆回购净投放量 |
| 英文代码 | `policy_repo_net` |
| 构建公式 | `Repo_Net(t) = 逆回购投放(t) - 逆回购到期(t)` |
| 业务释义 | 净投放为正，流动性宽松，利好债市；连续大幅净回笼是利空信号 |
| 预期方向 | **负相关**（净投放↑ → 流动性宽松 → 收益率↓） |

---

## 七、第六类：情绪面因子

> **逻辑基础**：市场情绪反映投资者的风险偏好，是短期价格波动的重要驱动因素。

### 7.1 国债期货持仓量因子

| 属性 | 说明 |
|------|------|
| 中文名 | 国债期货（30年）持仓量变化 |
| 英文代码 | `sentiment_futures_oi` |
| 构建公式 | `OI_Change(t) = OI(t) - OI(t-1)` |
| 业务释义 | 持仓量放大通常伴随机构加仓，反映做多情绪增强；减仓为利空信号 |
| 预期方向 | **负相关**（持仓增加 → 做多情绪强 → 收益率↓） |

---

### 7.2 股债跷跷板因子

| 属性 | 说明 |
|------|------|
| 中文名 | 权益市场滚动收益率（风险偏好代理） |
| 英文代码 | `sentiment_equity_return` |
| 构建公式 | `Equity_Return(t, W) = (CSI300(t) - CSI300(t-W)) / CSI300(t-W)`，W=4 |
| 业务释义 | 股市上涨时，风险偏好提升，资金从债市流向股市（跷跷板效应），收益率上行 |
| 预期方向 | **正相关**（权益市场上涨 → 债市资金流出 → 收益率↑） |

## 八、因子质量评价标准

在因子检验后，按以下标准对因子进行分级：

| 评级 | IC 均值 | IC_IR | 经济逻辑 | 处置建议 |
|------|---------|-------|----------|----------|
| ⭐⭐⭐ 优质因子 | ≥ 0.10 | ≥ 0.5 | 清晰 | 直接纳入因子库 |
| ⭐⭐ 潜力因子 | 0.05~0.10 | 0.3~0.5 | 清晰 | 纳入，关注稳定性 |
| ⭐ 弱效因子 | 0.03~0.05 | < 0.3 | 清晰 | 备选，结合组合使用 |
| ✗ 无效因子 | < 0.03 | 任意 | 任意 | 排除 |
| ⚠ 存疑因子 | 任意 | 任意 | 不清晰 | 谨慎使用，需进一步验证 |

## 九、基于用户提供数据的因子构建路径

当用户提供**机构净买入量**类数据时，优先构建以下因子组合（覆盖不同时间维度）：

| 因子代码 | 说明 | 时间维度 |
|----------|------|----------|
| `flow_net_*` | 当期净买入 | 即时信号 |
| `flow_ma4_*` | 4期滚动均值 | 短期趋势 |
| `flow_ma12_*` | 12期滚动均值 | 中期趋势 |
| `flow_mom_*` | 净买入环比变化 | 加速度信号 |
| `flow_cumsum12_*` | 近12期累积买入 | 累积效应 |
| `flow_composite` | 多机构综合 | 整体市场信号 |
| `flow_divergence` | 机构分歧度 | 不确定性信号 |

若数据中同时包含宏观数据，则补充宏观/资金面类因子，构成多维度因子体系。
