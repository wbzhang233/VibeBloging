## 一、资产负债管理与司库量化策略背景

### 1.1 银行资产负债管理的核心目标

银行资产负债管理部（ALM）负责管理全行的利率风险、流动性风险和资本效率。在利率市场化深化和利差收窄的背景下，司库量化策略正从"被动管理"向"主动创收"转变：

- **传统模式**：基于宏观判断，主观决定债券配置久期和仓位
- **量化模式**：基于系统性因子挖掘，构建规则化、可回测的交易策略

司库量化策略的核心竞争力来自：**对国债收益率变动方向的提前预判能力**。

### 1.2 30年期国债的市场特征

30年期超长国债是近年来机构投资者重点关注的品种，具有以下特征：

| 特征 | 说明 |
|------|------|
| **高久期敏感性** | DV01（每bp收益率变化的价格影响）显著高于短债，利率波动放大效应强 |
| **机构主导** | 主要持有者为保险（负债久期匹配需求）、银行自营（配置盘）、基金（交易盘） |
| **流动性分层** | 二级市场流动性弱于10年期，大单冲击明显，机构行为具有信号价值 |
| **政策敏感** | 对货币政策预期、监管政策（保险资本要求）敏感度高 |
| **供需驱动** | 财政部发行节奏、特别国债计划对供给端有重要影响 |

### 1.3 影响30年期国债收益率的核心因素

```
30年期国债收益率 = 无风险利率基准 + 期限溢价 + 机构行为溢价

影响维度：
┌─────────────────────────────────────────────────────┐
│ 宏观基本面：GDP增速、CPI、PMI → 决定利率中枢方向    │
├─────────────────────────────────────────────────────┤
│ 货币政策：MLF利率、降准 → 影响资金成本和预期        │
├─────────────────────────────────────────────────────┤
│ 资金面：DR007、超储率、银行间流动性 → 短端传导       │
├─────────────────────────────────────────────────────┤
│ 机构行为：保险/基金/券商净买入 → 直接驱动供需       │
├─────────────────────────────────────────────────────┤
│ 技术面：价格动量、均线偏离 → 反映市场情绪和趋势     │
└─────────────────────────────────────────────────────┘
```

### 1.4 机构投资者行为的信号价值

**保险资金**：
- 以负债久期匹配为核心诉求，超长债配置需求相对稳定
- 大幅增配信号：寿险保费高增、监管鼓励拉长久期 → 利好超长债
- 配置季节性：1月开门红资金部署期、年末会计结算期前

**基金（债券型）**：
- 以交易盘为主，对利率预期变化反应敏速
- 持续增配 → 市场利率下行预期强；赎回压力 → 卖盘压力

**券商自营**：
- 兼具配置和交易，对信用利差、流动性溢价敏感
- 超预期增配往往领先于市场方向

**核心逻辑**：机构净买入量的趋势和拐点往往领先于收益率变化 2-4 周，构成重要的先行指标。

## 二、量化策略研究框架

### 2.1 因子挖掘在策略构建中的位置

```
司库量化策略研究流程：

数据获取
    ↓
因子挖掘（本技能覆盖范围）
  ├── EDA 数据探索
  ├── 数据预处理
  ├── 特征工程（因子构建）
  └── 因子检验（筛选有效因子）
    ↓
策略构建
  ├── 因子合成（等权/IC加权/最优化）
  ├── 信号生成（阈值法/评分法）
  └── 仓位管理
    ↓
策略回测
  ├── 历史模拟
  ├── 风险指标计算
  └── 基准对比
    ↓
策略部署与监控
```

### 2.2 因子有效性的评价框架

一个有效的因子需要满足：

1. **统计显著性**：IC 均值在统计上显著不为零
2. **稳健性**：在不同时期、不同市场环境下表现稳定
3. **经济逻辑**：有清晰的业务机制解释
4. **独立性**：与已有因子相关性不过高（避免信息重叠）
5. **可实现性**：数据可获得，无未来数据泄露

## 三、EDA 分析详细流程指引

### 3.1 EDA 的核心目标

数据探索性分析（EDA）不是形式上的图表展示，而是要**发现数据的故事**：

| 分析维度 | 目的 | 后续影响 |
|----------|------|----------|
| 时序特征 | 了解数据的趋势性、季节性、结构性断点 | 决定是否需要差分、去趋势 |
| 分布特征 | 发现尖峰厚尾、偏态 | 决定异常值处理策略 |
| 相关性 | 初步识别与目标变量的关联 | 筛选有潜力的候选因子 |
| 缺失值 | 了解缺失模式（随机/系统性） | 决定填充方法 |
| 联动性 | 观察底层数据与目标变量的领先滞后关系 | 确定预测期 forward_period |

### 3.2 时序图解读要点

分析每个底层变量的时序图时，需要特别关注：

- **趋势性**：是否存在明显的上升/下降趋势（可能需要差分处理）
- **结构性断点**：是否有某个时间点出现数据的突然跳变（如统计口径变化、新政策实施）
- **季节性**：是否存在固定周期的波动（保险资金通常有年初集中部署的季节性）
- **极端值**：是否存在明显离群点（需判断是真实市场事件还是数据错误）

### 3.3 双轴联动图解读要点

底层变量与目标变量的双轴时序图，是 EDA 中最重要的图表之一：

- **领先关系**：底层变量的拐点是否早于目标变量的拐点？领先多少期？
- **同向性**：两者的趋势方向是否一致？在特定时期是否出现背离？
- **量级关系**：在波动剧烈的市场事件（如债市调整、宽松周期）中，联动关系是否增强？

### 3.4 相关性分析注意事项

- Pearson 相关系数假设线性关系；若散点图显示明显非线性，需补充 Spearman 秩相关
- 相关性不等于因果性；高相关可能是时期特异（spurious correlation）
- 在报告的 EDA 部分需明确说明：哪些变量与目标变量相关性强，并初步推断可能的业务机制

### 3.5 缺失值分析与处理决策

| 缺失比例 | 建议处理方式 |
|----------|-------------|
| < 5% | 前向填充或线性插值 |
| 5% ~ 20% | 分析缺失模式，选择填充；补充缺失说明 |
| > 20% | 评估该变量是否可用；如系统性缺失（如历史数据不可得），需标注数据起始时间 |
| 连续缺失 > 5期 | 不填充，保持 NaN；因子计算时该区间自动排除 |
