## 一、大模型与基础研究

### 1.1 OpenAI GPT-Rosalind：生命科学专属前沿推理模型

OpenAI 于 4 月 16 日正式发布 GPT-Rosalind，这是其首个面向生命科学领域的专项推理模型，以 DNA 双螺旋结构发现者罗莎琳德·富兰克林命名，颇具象征意义。该模型专为生物学、药物发现与转化医学工作流程优化，在化学反应机制推理、蛋白质突变效应预测、基因组学分析等核心科学子领域均达到最优水平。现实意义在于：一款新药从靶点发现到监管批准通常需要 10-15 年，GPT-Rosalind 的目标是压缩早期发现阶段的时间成本。它目前以研究预览形式向 ChatGPT、Codex 及 API 用户开放，合作伙伴包括 Amgen、Moderna、Allen Institute 和 Thermo Fisher。同步推出的生命科学 Codex 插件，可连接 50+ 科学工具与数据库，让科学家将模型能力直接嵌入现有研究工作流。💡 这标志着大模型正从通用推理向垂直科学应用的深度渗透，生物医药 AI 赛道的竞争将显著提速。

### 1.2 HBM 供应危机：内存厂商 2027 年底仅能满足 60% 需求

据 The Verge 4 月 18 日报道，内存行业分析人士预计，到 2027 年底，HBM（高带宽存储器）制造商仅能满足市场需求的 60%。AI 算力扩张的主要瓶颈已从芯片算力本身转移至存储带宽——每张 H100/B200 需要配套数量庞大的 HBM，而 SK 海力士、三星、美光的产能提升远未能跟上英伟达的出货节奏。📌 这一缺口将在 2026-2027 年持续制约 AI 数据中心的扩建速度，也意味着拥有 HBM 产能的供应商将长期处于卖方市场。

## 二、AI 产品与应用

### 2.1 Anthropic 推出 Claude Design：创意设计的 AI 协作新入口

4 月 17 日，Anthropic 旗下 Anthropic Labs 发布 Claude Design，由最新模型 Opus 4.7 驱动，定位为 AI 创意协作工具，支持用户生成产品设计、交互原型、演示幻灯片、营销物料等视觉内容。目前以研究预览版形式向付费订阅者开放。Claude Design 的意义在于：它试图让 Anthropic 的"思考能力"直接转化为可交付的视觉工作成果，而不只是文字。这与 Figma AI、Canva AI 等产品形成正面竞争，同时背靠 Anthropic 在安全性和推理能力上的积累，有望形成差异化。💡 AI 创意工具从"辅助写作"向"直接产出设计物料"的跃迁，正在重塑设计师的工作流程边界。

### 2.2 SimpleClosure：死亡公司数据正在成为 AI 训练的新矿藏

一家名为 SimpleClosure 的初创公司——专注于帮助企业有序关停——近期推出新工具，帮助倒闭公司将其遗留的代码库、Slack 消息、电子邮件和工作文档出售给需要训练数据的 AI 公司。Forbes 报道指出，这一需求催生了一个新兴行业："强化学习训练场"（RL Gym）——利用真实企业环境数据，构建 AI Agent 可以练习的模拟真实工作场景。📌 这暗示高质量的企业级真实数据正在成为稀缺资产。对于 AI Agent 的训练而言，"真实工作场景"比合成数据更有价值，这一趋势值得持续关注。

## 三、AI Agent 与智能体

### 3.1 OpenAI Codex 重大更新：Agent 获得"电脑操作权"

4 月 16 日，OpenAI 发布 Codex 重大版本更新，这是 Agent 能力跃升的里程碑事件。核心亮点：后台电脑使用（Background Computer Use）——Codex 可以在用户工作的同时，以独立光标在 macOS 上并行操作多个应用，无需 API 也能直接与任意软件交互。此外，Codex 新增内置浏览器，用户可以在页面上标注注释直接指挥 Agent；新增 gpt-image-1.5 图像生成能力；新增 90+ 插件覆盖 Atlassian Jira、CircleCI、GitLab、Microsoft Suite 等开发工具链；并引入记忆功能（Memory），可跨会话保留用户偏好、修正记录和项目上下文。最值得关注的是自主调度能力：Codex 可以为自己安排未来任务，在数天乃至数周内自动持续推进长期项目。💡 这是从"AI 副驾驶"向"AI 同事"的质的跨越，软件开发全生命周期的 Agent 自动化已具备雏形。

### 3.2 OpenAI Agents SDK 升级：标准化 Agent 基础设施

4 月 15 日，OpenAI 发布 Agents SDK 重大更新，重点解决从原型到生产的基础设施鸿沟。新版本引入：原生沙箱执行（支持 Blaxel、E2B、Modal、Cloudflare 等多家沙箱提供商）；Manifest 抽象层（统一描述 Agent 工作区，兼容 AWS S3/GCS/Azure Blob 等存储）；内置支持 MCP、AGENTS.md、Skills 等新兴 Agent 协议标准；以及配置化记忆管理。安全设计上将 harness 与代码执行环境分离，防止模型生成代码访问宿主凭证，大幅降低提示注入和数据泄露风险。💡 OpenAI 正在用标准化基础设施锁定 Agent 开发者生态，与 LangChain、LlamaIndex 等框架的竞争进入新阶段。

### 3.3 Google 强化 Android AI Agent 开发基础设施

4 月 16 日，Google 宣布推出 Android Skills GitHub 仓库和 Android Knowledge Base，为 AI Agent 提供执行 Android 开发任务所需的知识和资源框架，同步升级 Android CLI。这一布局意义在于：AI Agent 正在系统性地进入移动端开发场景，Google 试图通过标准化的"技能知识库"降低 Agent 开发门槛，推动 Android AI Agent 生态的快速成熟。

## 四、AI 基础设施与安全

### 4.1 Anthropic Mythos：网络安全专属模型进入美国国安机构

Anthropic 的私密网络安全模型 Mythos Preview 的扩散速度超出预期。该模型已被 Nvidia、Apple、JPMorgan Chase 等大型机构用于检测高危系统漏洞。更引人注目的是：NSA 据报已获得 Mythos 访问权限，尽管该机构此前被标注为"供应链风险"；白宫也在准备正式接入方案。这背后是 Anthropic 与 Project Glasswing 的深度布局——该项目已集结 AWS、Apple、Broadcom、Cisco、CrowdStrike、Google、JPMorgan、Linux Foundation、Microsoft、Nvidia、Palo Alto Networks 等重量级机构，共同构建"全球关键软件安全"防线。💡 AI 的安全军事化应用已从讨论进入落地，Anthropic 的"安全优先"品牌正在成为其进入政府和国防市场的通行证。

### 4.2 Google Gemini 正与五角大楼洽谈涉密系统准入

The Verge 4 月 16 日报道，Google 正与美国国防部谈判，希望将 Gemini 的使用范围从当前的"非涉密场景"扩展至涉密环境。Google 提出的合同条款与 OpenAI 此前签署的协议类似，均包含"用于所有合法目的"的宽泛授权。法律学者指出，OpenAI 协议中看似限制"完全自主致命武器"和"大规模国内监控"的条款，在"所有合法目的"框架下实际约束力存疑。📌 AI 巨头争夺军政市场的竞争进入白热化，这既是商业机会，也是公众对 AI 军事化使用监管的重大隐忧。

### 4.3 美国数据中心将强制提交能源使用报告

美国能源信息署（EIA）计划向全国数据中心推行强制性能源使用调查，已在德州、华盛顿州、北弗吉尼亚等数据中心密集地区完成试点。此举回应了参议员沃伦、霍利等两党议员的立法推动，旨在摸清 AI 基础设施的能源消耗底数。这是美国政府首次系统性推进数据中心能耗透明化，为后续碳排放监管和电网规划提供数据基础。

## 五、国内与全球 AI 治理动态

### 5.1 AI 政治化：AI 舆论账号大规模渗透美国社交媒体

《纽约时报》调查发现，Instagram、TikTok 和 Facebook 上出现数百个 AI 生成的假账号，配合统一话术为特朗普保守派议程造势，被认为是中期选举前的舆论预热动作。这些账号由内容农场或营销公司批量创建，使用相同措辞和图像风格，专家认为 AI Avatar 的批量生产成本已极低。💡 这一现象揭示了 AI 在信息战中的危险潜力——大规模、低成本的"AI 舆论军队"不再是科幻，已是现实。平台治理和身份认证机制面临前所未有的挑战。

### 5.2 Sanders 联合工会警告：AI 将终结制造业岗位

参议员伯尼·桑德斯联合跨行业工会领袖举行集会，要求对 AI 扩张设立就业保护措施，并正式提出数据中心建设暂停法案（与 AOC 联署）。Sanders 警告：若不加约束，十年内"制造业岗位"的概念将不复存在。这是美国政界对 AI 就业冲击最强硬的政策表态之一，预示着 AI 监管的政治阻力正在系统性积聚。
